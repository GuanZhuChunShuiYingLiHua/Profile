2. 核心任务目标
      1. 技术文档清洗（针对吉利项目）
      * 利用 LLM 辅助完成高质量三元组的清洗与提取。
      * 重点处理整车级汽车技术文档。
     2. LLM 增强图谱构建（针对国创项目）
      * 探索 Prompt Engineering 提升抽取效率的路径。
      * 进行 RAG (检索增强生成) 架构验证。
      * 开发实用系统，并完成 1 篇高水平论文。
     3. 多模态关联应用（针对剧目项目）
      * 实现图片、视频、文本的多模态数据关联。
      * 落地推荐系统、智能检索及内容辅助决定。

 * 3 月： 完成三个项目的本体结构（Ontology）设计；完成吉利项目对接（确定大纲与逻辑）。

 * 8 月： 项目最终交付。
